## 사용자 최종 확인 항목 (릴리스 전)

1. 디스코드에서 아무 커스텀 이모지 우클릭 → 링크 복사 → 피커 ＋에 붙여넣기 등록
   (실 CDN 다운로드 경로 — 개발 검증은 mock으로만 수행)
2. 등록 항목 클릭 → 디스코드 입력창에 첨부되는지 → Enter 전송
3. 트레이 메뉴 육안 확인(한국어 표기, 핫키 라디오) + "종료"로 완전 종료(작업관리자)

## 개발 중 발견·수정된 통합 이슈 (재발 방지 기록)

- 64비트 ctypes: argtypes 없이 `HWND_TOPMOST(-1)` 전달 시 0xFFFFFFFF로 잘려
  SetWindowPos 무효 → HWND argtypes 명시로 해결
- 크로스스레드 SetWindowPos 동기 메시지 교착 → `SWP_ASYNCWINDOWPOS`
- pywebview js_api: 공개 속성을 재귀 노출하므로 컨트롤러/라이브러리 참조는
  반드시 `_` 프라이빗 (순환 그래프 → 브리지 생성 무한 대기)
- WebView2는 file:// 서브리소스 차단 → 127.0.0.1 AssetServer(id 조회)로 서빙
- DPI: 프로세스 Per-Monitor aware 설정 + GetDpiForMonitor로 창 크기 산출
